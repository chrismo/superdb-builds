# SuperDB Builds

Independent builds of the `super` binary from the
[BSD-3-Clause licensed source](https://github.com/chrismo/super).

Brimdata [changed the upstream license](https://github.com/brimdata/super/commit/e2e341f8b2559e640ca1e7f8a0b42078ddfef01c)
from BSD-3-Clause to a restrictive source-available license. This repo builds
from a fork pinned to the last open-source commit.

The [asdf-superdb plugin](https://github.com/chrismo/asdf-superdb) installs
binaries from the releases here.
